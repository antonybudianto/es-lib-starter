# ES Lib Starter

Template starter for developing ES Library with TypeScript

## Development

Link your library for local use

```
npm link

# change dir to your app that will use this lib, and run:
npm link <my-lib>
```

## Test

```
npm test
```

## Build

```
npm run build
```

## Publish

```
npm publish
```

or using [np](https://github.com/sindresorhus/np)

```
npm run release
```
